== Description ==

BTCQuote is an open-source, real-time Bitcoin price widget for your website or blog.

This plugin for Wordpress provides an easy way to embed the [BTCQuote](http://btcquote.com) widget in your Wordpress site.

== Installation ==

1. Upload `btcquote.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('btcquote'); ?>` in your templates
